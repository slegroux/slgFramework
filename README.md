#  slgFramework
Sylvain Le Groux 2013


## About

A repository of cross-platform C++ classes for real-time audio, midi and graphics processing

* slgAudio
* slgFx
* slgSpectrum
* slgMidi

## Dependencies

* RtAudio
* RtMidi
* chuck_fft

## Compilation
In each slgName subdirectory

make

./test
